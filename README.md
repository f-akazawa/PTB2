Psychotoolbox2(4)
======================
本家がバージョン３だったのでプロジェクト名は４にすれば良かった・・・  
Matlab toolboxの素晴らしいツールであるPTB３のデモをOculus goに移植してます。  
 
使い方
------
### Oculus goの設定 ###
デベロッパーサイトで開発者モードをオンにして、PCをUSBで接続した後USBデバッグを許可してください  
 
デモの解説
---------------- 
+   `DotDemo` :
    DotDemoの移植
 
+   `Drift` :
    Driftの移植版

+   `RandomDots` : 
    DrawDot3Dの移植版

随時追加予定
